# sarthi

A cross-platform app that serves as a personal spiritual guide, allowing users to ask life’s questions and receive thoughtful, AI-generated answers rooted in the teachings of the Bhagavad Gita, Quran, or Bible, with scripture-specific context, verse references, and a fully private, login-free experience stored entirely on the user’s device.
